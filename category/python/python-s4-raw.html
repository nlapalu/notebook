## Execution du code temps/mémoire

### Calculer le temps d'éxecution

Si vous êtes sous linux et que vous souhaitez simplement tester le temps d'execution d'un script, le plus simple est d'utiliser la commande system `time`.

_time.py:_
```python
#!/usr/bin/env python3

import time
time.sleep(5)
```
execute:
```sh
time ./time.py

real    0m5.050s
user    0m0.036s
sys     0m0.008s
```

Pour un affichage plus complet:
```sh
/usr/bin/time -v ./time.py
```

Si vous souhaitez mesurer le temps d'execution de seulement une partie de votre code, il existe plusieurs modules qui vous permettront de faire cela.
